`ironclaw-zh` 是基於 [nearai/ironclaw](https://github.com/nearai/ironclaw) 的社群維護發行版，重點面向中文體驗、本地優先部署與對上游安全架構的保留。

> 本專案為社群維護版本，不是 NEAR AI 官方發行版。

## 簡介

- 基於 IronClaw 的中文發行版。
- 面向本地優先和自託管使用情境。
- 提供中文文件、Web UI 本地化、預設與技能支援。
