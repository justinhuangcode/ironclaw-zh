`ironclaw-zh` is a community-maintained distribution based on [nearai/ironclaw](https://github.com/nearai/ironclaw). It focuses on Chinese user experience, local-first deployment, and preserving IronClaw's security-oriented architecture.

`ironclaw-zh` 是基于 [nearai/ironclaw](https://github.com/nearai/ironclaw) 的社区维护发行版，重点面向中文体验、本地优先部署和对上游安全架构的保留。

> This project is community-maintained and is not an official NEAR AI release.
>
> 本项目为社区维护版本，不是 NEAR AI 官方发行版。

## Overview

- Chinese-localized distribution based on IronClaw.
- Built for local-first and self-hosted usage.
- Includes Chinese documentation, Web UI localization, presets, and skills support.
