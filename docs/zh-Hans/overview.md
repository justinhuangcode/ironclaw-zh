# 项目概览

`ironclaw-zh` 是基于 [nearai/ironclaw](https://github.com/nearai/ironclaw) 的中文本地化、本地优先发行版，面向希望安全自托管的中文用户。

> 本项目为社区维护版本，不是 NEAR AI 官方发行版。

## 适合谁

- 中文技术用户与自托管用户
- 在意隐私、本地控制和安全边界的用户
- 希望在自己的设备或服务器上运行 AI assistant 的用户

## 主要内容

- 中文 README 与中文文档
- 中文 Web UI 文案与 locale 结构
- 中文默认提示与中文技能
- 更适合中文用户的部署与配置说明
- 更适合本地部署的预设和发行包装

## 与上游的关系

上游项目为 [nearai/ironclaw](https://github.com/nearai/ironclaw)。

本项目会持续跟随上游版本，通用修复会尽量优先回提上游。
