# 术语表

- OpenClaw
- WASM

## 核心产品词

- self-hosting -> 自托管
- local-first -> 本地优先
- distribution -> 发行版
- gateway -> 网关
- workspace -> 工作区
- skills -> 技能
- sandbox -> 沙箱
- extensions -> 扩展
- logs -> 日志
- jobs -> 任务
- routines -> 例程

## 用户可见层词汇

UI 层的 `thread` 统一翻译为：

- 会话

而不是“线程”。

`conversation` 统一翻译为：

- 对话
